# web-Omni-
高效的浏览器插件，支持隐私保护与广告拦截等




## 局域网传输使用说明

Web-Omni 现已支持桌面端与手机端之间的局域网文件传输，提供两种连接方式：

- 模式 A：下载手机页面
- 模式 B：导出局域网页包 ZIP

### 使用前提

- 手机和电脑连接到同一个 Wi-Fi
- 手机请使用系统浏览器或主流浏览器打开页面
- 传输基于 WebRTC，建立连接时仍需要信令 / STUN 网络可用
- Windows 首次启动本地网页服务时，如果弹出防火墙提示，请允许专用网络访问

---

### 模式 A：下载手机页面

适合快速使用。

#### 步骤

1. 在电脑端打开 `局域网传输`
2. 记录页面上的房间码
3. 点击 `下载手机页面`
4. 将下载好的 HTML 文件发送到手机
5. 在手机浏览器中打开该 HTML 文件
6. 输入房间码并点击连接
7. 连接成功后即可在手机和电脑之间互传文件

#### 注意

- 不要使用聊天软件内置预览打开 HTML
- 如果电脑端房间码重新生成，需要重新输入新房间码
- 如果手机打开的是旧页面，请重新下载最新版 HTML

---

### 模式 B：导出局域网页包 ZIP

适合连接更稳定的使用方式，推荐手机端优先使用此模式。

#### 步骤

1. 在电脑端打开 `局域网传输`
2. 点击 `导出局域网页包 ZIP`
3. 将下载得到的 `web-omni-mobile-kit.zip` 解压
4. 双击运行 `start-mobile-server.cmd`
5. PowerShell 窗口会显示一个或多个地址，例如：
   `http://192.168.1.23:8787/index.html`
6. 在手机浏览器中打开能访问的那个地址
7. 输入房间码或使用页面预填房间码进行连接
8. 保持 PowerShell 窗口打开，传输完成后再关闭

#### 压缩包内容

- `index.html`：手机端页面
- `start-mobile-server.cmd`：Windows 启动脚本
- `serve-mobile.ps1`：本地网页服务脚本
- `README.txt`：简要说明

---

### 文件传输说明

#### 电脑发送到手机

1. 手机连接成功
2. 电脑端拖拽文件到传输区域，或点击选择文件
3. 手机端接收完成后自动保存 / 下载

#### 手机发送到电脑

1. 手机连接成功
2. 手机端点击选择文件
3. 电脑端接收完成后自动保存 / 下载

---

### 已知说明

- 连接建立依赖 WebRTC，因此局域网环境下仍可能受到网络策略影响
- 暂未内置 TURN 服务，复杂网络环境下连接成功率可能受限
- 如果房间码变化，导出的手机页面或局域网页包建议重新生成
- 局域网页包模式下，请始终通过浏览器打开地址，不要直接预览文件内容
